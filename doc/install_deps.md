# How to install dependency pacages

**Fedora**

I tested on Fedora Linux 42.

```
$ sudo dnf install \
  clang-tools-extra \
  cairo-devel \
  pango-devel \
  wayland-devel \
  libxkbcommon-devel \
  scdoc
```

**Ubuntu**

I tested on Ubuntu 24.04 noble.

```
$ sudo apt install \
  clang-format \
  libcairo2-dev \
  libpango1.0-dev \
  libwayland-dev \
  libxkbcommon-dev \
  scdoc
```
